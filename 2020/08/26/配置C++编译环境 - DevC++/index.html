配置C++编译环境 - DevC++

本博客所有文章除特別声明外,均采用CC BY 4.0许可协议。
若需转载请先通知作者,并注明来源,谢谢!

1. 前言

DevC++是一个Windows环境下的一个适合于初学者使用的轻量级C/C++开发环境(内存小是小,就是界面有点丑)。它集合了MinGW中的GCC编译器,GDB调试器以及AStyle格式整理器(AStyle格式整理器让人省了很多力气,只要按下Ctrl+Shift+A就可以自动整理格式)等众多自由软件。

2. 下载

下载传送门1:softonic.com
下载传送门2:pc.qq.com
读者可自行选择下载链接。

3. 安装

双击安装文件,会弹出一个窗口:

正在加载安装程序中,等几秒就好了。

接下来会让你选择安装的语言,这里不能选择中文,所以我就选了英文(反正后面也可以改)。

然后点击”I Agree”(同意条款),

在上面选择”Full”,

最后选择安装路径后,按下”Install”就开始安装了。

安装完毕后,点击”Finish”就大功告成了!

4. 初步配置Devc++

打开软件会出现一个窗口,如下图所示:

在右侧选择语言,英语不行的读者还是就选择简体中文吧。
然后选择字体与风格,这里保持默认(反正自带的风格都是丑的不行,在后面的”配置C++编译环境 - DevC++(2)”中我会讲述如何配置稍微好看一点点的主题),点击”Next”即可。

最够点击OK即可进入Devc++,开始编写代码。
